Style GAN2

This is a PyTorch implementation of the paper Analyzing and Improving the Image Quality of StyleGAN which introduces Style GAN2. Style GAN2 is an improvement over Style GAN from the paper A Style-Based Generator Architecture for Generative Adversarial Networks. And Style GAN is based on Progressive GAN from the paper Progressive Growing of GANs for Improved Quality, Stability, and Variation. All three papers are from same set of authors from NVIDIA AI.

Our implementation is a minimalistic Style GAN2 model training code. Only single GPU training is supported to keep the implementation simple. We managed to shrink it to keep it at a little more than 350 lines of code.

We’ll first introduce the three papers at a high level.

Generative Adversarial Networks

Generative adversarial networks have two components; the generator and the discriminator. The generator network takes a random latent vector ($z \in \mathcal{Z}$) and tries to generate a realistic image. The discriminator network tries to differentiate the real images from generated images. When we train the two networks together the generator starts generating images indistinguishable from real images.

Progressive GAN

Progressive GAN generate high-resolution images ($1080 \times 1080$) of size. It does so by progressively increasing the image size. First, it trains a network that produces a $4 \times 4$ image, then $8 \times 8$ , then an $16 \times 16$ image, and so on upto the desired image resolution.

At each resolution the generator network produces an image in latent space which is converted into RGB, with a $1 \times 1$ convolution. When we progress from a lower resolution to a higher resolution (say from $4 \times 4$ to $8 \times 8$ ) we scale the latent image by $2\times$ and add a new block (two $3 \times 3$ convolution layers) and a new $1 \times 1$ layer to get RGB. The transition is done smoothly by adding a residual connection to the $2\times$ scaled $4 \times 4$ RGB image. The weight of this residual connection is slowly reduced, to let the new block to take over.

The discriminator is a mirror image of the generator network. The progressive growing of the disciminator is done similarly.

They use minibatch standard deviation to increase variation and equalized learning rate which we discussed below in the implementation. They also use pixel-wise normalization where at each pixel the feature vector is normalized. They apply this to all the convlution layer outputs (except RGB).

Style GAN

Style GAN improves the generator of Progressive GAN keeping the discriminator architecture same.

Mapping Network

It maps the random latent vector ($z \in \mathcal{Z}$)i nto a different latent space ($w \in \mathcal{W}$), with a 8-layer neural network. This gives a intemediate latent space $\mathcal{W}$ where the factors of variations are more linear (disentangled).

AdaIN

Then $w$ is transformed into two vectors (styles) per layer, $i$, $y_i = (y_{s,i}, y_{b,i}) = f_{A_i}(w)$ and used for scaling and shifting (biasing) in each layer with $\text{AdaIN}$ operator (normalize and scale):

Style Mixing

To prevent the generator from assuming adjacent styles are correlated, they randomly use different styles for different blocks. That is, they sample two latent vectors $(z_1, z_2)$ and corresponding $(w_1, w_2)$ and use $w_1$ based styles for some blocks and $w_2$ based styles for some blacks randomly.

Stocastic Variation

Noise is made available to each block which helps generator create more realistic images. Noise is scaled per channel by a learned weight.

Bilinear Up and Down Sampling

All the up and down sampling operations are accompanied by bilinear smoothing.

Style GAN 2

Style GAN 2 changes both the generator and the discriminator of Style GAN.

Weight Modulation and Demodulation

They remove the $\text{AdaIN}$ operator and replace it weight modulation and demodulation step. This is supposed to improve what they call droplet artifacts that are present in generated images, which are caused by the normalization in $\text{AdaIN}$ operator. Style vector per layer is calculated from $w_i \in \mathcal{W}$ as $s_i = f_{A_i}(w_i)$.

Then the convolution weights $w$ are modulated as follows. ($w$ here on refers to weights not intermediate latent space, we are sticking to the same notation as the paper.) Then it’s demodulated by normalizing, where $i$ is the input channel, $j$ is the output channel, and $k$ is the kernel index.

Path Length Regularization

Path length regularization encourages a fixed-size step in $\mathcal{W}$ to result in a non-zero, fixed-magnitude change in generated image.

No Progressive Growing

StyleGAN2 uses residual connections (with downsampling) in the discriminator and skip connections in the generator with upsampling (the RGB outputs from each layer are added - no residual connections in feature maps). They show that with experiemnts that the contribution of low resolution layers is higher at beginning of the training and then high-resolution layers take over.

103class MappingNetwork(nn.Module):
# 
104    def __init__(self, features: int, n_layers: int):
105        super().__init__()
106
107        layers = []
108        for i in range(n_layers):
109            layers.append(EqualizedLinear(features, features, lr_mul=1.))
110            layers.append(nn.LeakyReLU(negative_slope=0.2, inplace=True))
111
112        self.net = nn.Sequential(*layers)
# 
114    def forward(self, x: torch.Tensor):
115        x = F.normalize(x, dim=1)
116        return self.net(x)

119class Discriminator(nn.Module):
# 
123    def __init__(self, n_layers, n_image_features=3, n_features=32, max_features=512):
124        super().__init__()
125
126        self.from_rgb = nn.Sequential(
127            EqualizedConv2d(n_image_features, n_features, 1),
128            nn.LeakyReLU(0.2, True),
129        )
130
131        blocks = []
132        out_features = n_features
133        for i in range(n_layers):
134            in_features = out_features
135            out_features = min(out_features * 2, max_features)
136            blocks.append(DiscriminatorBlock(in_features, out_features))
137
138        self.blocks = nn.Sequential(*blocks)
139
140        self.conv = EqualizedConv2d(out_features, out_features, 4)
141        self.logits = EqualizedLinear(out_features, 1)
# 
143    def __call__(self, x: torch.Tensor):
144        x = self.from_rgb(x)
145        x = self.blocks(x)
146
147        x = self.conv(x)
148        x = x.reshape(x.shape[0], -1)
149        x = self.logits(x)
150        return x
# 
153class DiscriminatorBlock(nn.Module):
# 
154    def __init__(self, in_features, out_features):
155        super().__init__()
156        self.residual = EqualizedConv2d(in_features, out_features, kernel_size=1, stride=2)
157
158        self.block = nn.Sequential(
159            EqualizedConv2d(in_features, in_features, kernel_size=3, padding=1),
160            nn.LeakyReLU(0.2, True),
161            EqualizedConv2d(in_features, out_features, kernel_size=3, padding=1),
162            nn.LeakyReLU(0.2, True),
163        )
164
165        self.down_sample = DownSample(out_features)
166        self.scale = 1 / math.sqrt(2)
# 
168    def forward(self, x):
169        residual = self.residual(x)
170        x = self.block(x)
171        x = self.down_sample(x)
172
173        return (x + residual) * self.scale
# 

176class Generator(nn.Module):
# 
180    def __init__(self, n_layers, d_latent, n_features=32, max_features=512):
181        super().__init__()
182
183        blocks = []
184        in_features = n_features
185        for i in range(n_layers):
186            out_features = in_features
187            in_features = min(in_features * 2, max_features)
188            blocks.append(GeneratorBlock(d_latent, in_features, out_features))
189
190        self.blocks = nn.ModuleList(list(reversed(blocks)))
191
192        self.up_sample = UpSample()
193        self.initial_constant = nn.Parameter(torch.randn((1, in_features, 4, 4)))
# 
195    def forward(self, styles: torch.Tensor, input_noise: torch.Tensor):
196        batch_size = styles.shape[0]
197
198        x = self.initial_constant.expand(batch_size, -1, -1, -1)
199
200        rgb = None
201
202        for i in range(len(self.blocks)):
203            if i != 0:
204                x = self.up_sample(x)
205
206            x, rgb = self.blocks[i](x, rgb, styles[:, i], input_noise)
207
208        return rgb
# 
211class GeneratorBlock(nn.Module):
# 
212    def __init__(self, latent_dim, in_features, out_features):
213        super().__init__()
214        self.to_style1 = EqualizedLinear(latent_dim, in_features)
215        self.to_noise1 = EqualizedLinear(1, out_features)
216        self.conv1 = Conv2dWeightModulate(in_features, out_features, kernel_size=3)
217
218        self.to_style2 = EqualizedLinear(latent_dim, out_features)
219        self.to_noise2 = EqualizedLinear(1, out_features)
220        self.conv2 = Conv2dWeightModulate(out_features, out_features, kernel_size=3)
221
222        self.activation = nn.LeakyReLU(0.2, True)
223        self.to_rgb = ToRGB(latent_dim, out_features)
# 
225    def forward(self, x: torch.Tensor, prev_rgb: torch.Tensor, istyle: torch.Tensor, inoise: torch.Tensor):
226        inoise = inoise[:, :x.shape[2], :x.shape[3], :]
227        noise1 = self.to_noise1(inoise).permute((0, 3, 2, 1))
228        noise2 = self.to_noise2(inoise).permute((0, 3, 2, 1))
229
230        style1 = self.to_style1(istyle)
231        x = self.conv1(x, style1)
232        x = self.activation(x + noise1)
233
234        style2 = self.to_style2(istyle)
235        x = self.conv2(x, style2)
236        x = self.activation(x + noise2)
237
238        rgb = self.to_rgb(x, prev_rgb, istyle)
239        return x, rgb
# 
242class ToRGB(nn.Module):
# 
243    def __init__(self, latent_dim, in_features):
244        super().__init__()
245        self.input_channel = in_features
246        self.to_style = EqualizedLinear(latent_dim, in_features)
247
248        self.conv = Conv2dWeightModulate(in_features, 3, kernel_size=1, demodulate=False)
249        self.up_sample = UpSample()
# 
251    def forward(self, x, prev_rgb, istyle):
252        style = self.to_style(istyle)
253        rgb = self.conv(x, style)
254
255        if prev_rgb is not None:
256            rgb = rgb + self.up_sample(prev_rgb)
257
258        return rgb

261class Conv2dWeightModulate(nn.Module):
# 
262    def __init__(self, in_features, out_features, kernel_size, demodulate=True, lr_mul=1., eps=1e-8):
263        super().__init__()
264        self.filters = out_features
265        self.demodulate = demodulate
266        self.kernel = kernel_size
267        self.padding = (self.kernel - 1) // 2
268
269        he_std = 1 / math.sqrt(in_features * kernel_size * kernel_size)
270        self.weight = nn.Parameter(torch.randn((out_features, in_features, kernel_size, kernel_size)) / lr_mul)
271        self.eps = eps
272
273        self.runtime_coef = lr_mul * he_std
# 
275    def forward(self, x, style):
276        b, c, h, w = x.shape
277
278        w1 = style[:, None, :, None, None]
279        w2 = self.weight[None, :, :, :, :] * self.runtime_coef
280        weights = w2 * (w1 + 1)
281
282        if self.demodulate:
283            d = torch.rsqrt((weights ** 2).sum(dim=(2, 3, 4), keepdim=True) + self.eps)
284            weights = weights * d
285
286        x = x.reshape(1, -1, h, w)
287
288        _, _, *ws = weights.shape
289        weights = weights.reshape(b * self.filters, *ws)
290
291        x = F.conv2d(x, weights, padding=self.padding, groups=b)
292
293        x = x.reshape(-1, self.filters, h, w)
294
295        return x

298class DownSample(nn.Module):
# 
299    def __init__(self, features: int):
300        super().__init__()
301        self.smooth = Smooth()
302        self.conv = EqualizedConv2d(features, features, kernel_size=3, padding=1, stride=2)
# 
304    def forward(self, x: torch.Tensor):
305        return self.conv(self.smooth(x))

308class UpSample(nn.Module):
# 
309    def __init__(self):
310        super().__init__()
311        self.up_sample = nn.Upsample(scale_factor=2, mode='bilinear', align_corners=False)

318class Smooth(nn.Module):
# 
319    def __init__(self):
320        super().__init__()
321        kernel = [[1, 2, 1],
322                  [2, 4, 2],
323                  [1, 2, 1]]
324        kernel = torch.tensor([[kernel]], dtype=torch.float)
325        kernel /= kernel.sum()
326        self.kernel = nn.Parameter(kernel, requires_grad=False)
# 
328    def forward(self, x: torch.Tensor):
329        b, c, h, w = x.shape
330        x = x.view(-1, 1, h, w)
331
332        x = F.conv2d(x, self.kernel, padding=1)
333
334        return x.view(b, c, h, w)

337class EqualizedLinear(nn.Module):
# 
338    def __init__(self, in_features: int, out_features: int, lr_mul: float = 1.):
339        super().__init__()
340        self.weight = nn.Parameter(torch.randn(out_features, in_features) / lr_mul)
341        self.bias = nn.Parameter(torch.zeros(out_features))
342
343        he_std = 1 / math.sqrt(in_features)
344        self.runtime_coef = lr_mul * he_std
345        self.lr_mul = lr_mul
# 
347    def __call__(self, x: torch.Tensor):
348        return F.linear(x, self.weight * self.runtime_coef, bias=self.bias * self.lr_mul)
# 
351class EqualizedConv2d(nn.Module):
# 
352    def __init__(self, in_features: int, out_features: int,
353                 kernel_size: int, padding: int = 0, stride: int = 1,
354                 lr_mul: float = 1.):
355        super().__init__()
356        self.stride = stride
357        self.padding = padding
358        self.weight = nn.Parameter(torch.randn((out_features, in_features, kernel_size, kernel_size)) / lr_mul)
359        self.bias = nn.Parameter(torch.zeros(out_features))
360
361        he_std = 1 / math.sqrt(in_features * kernel_size * kernel_size)
362        self.runtime_coef = lr_mul * he_std
363        self.lr_mul = lr_mul
# 
365    def forward(self, x: torch.Tensor):
366        return F.conv2d(x, self.weight * self.runtime_coef, bias=self.bias * self.lr_mul,
367                        padding=self.padding, stride=self.stride)

370class PathLengthPenalty(nn.Module):
# 
371    def __init__(self, beta: float):
372        super().__init__()
373        self.beta = beta
374        self.steps = nn.Parameter(torch.tensor(0.), requires_grad=False)
375        self.moving_average = nn.Parameter(torch.tensor(0.), requires_grad=False)
# 
377    def forward(self, styles: torch.Tensor, images: torch.Tensor):
378        device = images.device
379        num_pixels = images.shape[2] * images.shape[3]
380        pl_noise = torch.randn(images.shape, device=device) / math.sqrt(num_pixels)
381        outputs = (images * pl_noise).sum()
382
383        gradients, *_ = torch.autograd.grad(outputs=outputs,
384                                            inputs=styles,
385                                            grad_outputs=torch.ones(outputs.shape, device=device),
386                                            create_graph=True)
387
388        path_lengths = (gradients ** 2).sum(dim=2).mean(dim=1).sqrt()
389
390        if self.steps > 0:
391            a = self.moving_average / (1 - self.beta ** self.steps)
392            loss = torch.mean((path_lengths - a) ** 2)
393        else:
394            loss = path_lengths.new_tensor(0)
395
396        mean = path_lengths.mean().detach()
397
398        if not torch.isnan(mean):
399            self.moving_average.mul_(self.beta).add_(mean, alpha=1 - self.beta)
400            self.steps.add(1.)
401
402        return loss

474    def init(self):
475        self.dataset = Dataset(self.image_size)
476        dataloader = torch.utils.data.DataLoader(self.dataset, batch_size=self.batch_size,
477                                                 num_workers=32,

479        self.loader = cycle(dataloader)
480        self.n_layers = int(math.log2(self.image_size)) - 1
481
482        self.discriminator = Discriminator(self.n_layers - 1).to(self.device)
483        self.generator = Generator(self.n_layers, self.d_latent).to(self.device)
484        self.mapping_network = MappingNetwork(self.d_latent, self.mapping_network_layers).to(self.device)

487        self.discriminator_loss = DiscriminatorLoss()
488        self.generator_loss = GeneratorLoss()

490        self.discriminator_optimizer = torch.optim.Adam(
491            self.discriminator.parameters(),
492            lr=self.learning_rate, betas=self.betas
493        )
494        self.generator_optimizer = torch.optim.Adam(
495            self.generator.parameters(),
496            lr=self.learning_rate, betas=self.betas
497        )
498        self.mapping_network_optimizer = torch.optim.Adam(
499            self.mapping_network.parameters(),
500            lr=self.mapping_network_learning_rate, betas=self.betas
501        )
502        tracker.set_image("generated", True)
# 
504    def z_noise(self, batch_size):
505        return torch.randn(batch_size, self.d_latent).to(self.device)
# 
507    def z_to_w(self, z: torch.Tensor):
508        w = self.mapping_network(z)
509        return w[:, None, :].expand(-1, self.n_layers, -1)
# 
511    def image_noise(self, batch_size):
512        return torch.rand(batch_size, self.image_size, self.image_size, 1, device=self.device)
# 
514    def generate_images(self, batch_size):
515        z_style = self.z_noise(batch_size)
516        w_style = self.z_to_w(z_style)
517        noise = self.image_noise(batch_size)
518
519        generated_images = self.generator(w_style, noise)
520
521        return generated_images, w_style
# 
523    def step(self, idx):
#

Train Discriminator

525        self.discriminator_optimizer.zero_grad()
526
527        for i in range(self.gradient_accumulate_steps):
528            generated_images, _ = self.generate_images(self.batch_size)
529            fake_output = self.discriminator(generated_images.detach())
530
531            x = next(self.loader).to(self.device)
532            x.requires_grad_()
533            real_output = self.discriminator(x)
534
535            real_loss, fake_loss = self.discriminator_loss(real_output, fake_output)
536            disc_loss = real_loss + fake_loss
537
538            if (idx + 1) % self.lazy_gradient_penalty_interval == 0:
539                gp = self.gradient_penalty(x, real_output)
540                tracker.add('loss.gp', gp)
541                disc_loss = disc_loss + gp
542
543            disc_loss.backward()
544
545            tracker.add('loss.discriminator', disc_loss)
546
547        self.discriminator_optimizer.step()
#

Train Generator & Mapping Network

550        self.generator_optimizer.zero_grad()
551        self.mapping_network_optimizer.zero_grad()
552
553        for i in range(self.gradient_accumulate_steps):
554            generated_images, w_style = self.generate_images(self.batch_size)
555            fake_output = self.discriminator(generated_images)
556
557            gen_loss = self.generator_loss(fake_output)
558
559            if idx > self.lazy_path_penalty_after and (idx + 1) % self.lazy_path_penalty_interval == 0:
560                ppl = self.path_length_penalty(w_style, generated_images)
561                if not torch.isnan(ppl):
562                    gen_loss = gen_loss + ppl
563
564            gen_loss.backward()
565
566            tracker.add('loss.generator', gen_loss)
567
568        self.generator_optimizer.step()
569        self.mapping_network_optimizer.step()
570
571        if (idx + 1) % self.log_generated_interval == 0:
572            tracker.add('generated', generated_images)
573        if (idx + 1) % self.save_checkpoint_interval == 0:
574            experiment.save_checkpoint()
575
576        tracker.save()
# 
578    def run(self):
579        for i in monit.loop(150_000):
580            self.step(i)
581            if (i + 1) % self.log_generated_interval == 0:
582                tracker.new_line()
