Style GAN2

This is a PyTorch implementation of the paper Analyzing and Improving the Image Quality of StyleGAN which introduces Style GAN2. Style GAN2 is an improvement over Style GAN introduced in the paper A Style-Based Generator Architecture for Generative Adversarial Networks. And Style GAN is based on Progressive GAN from the paper Progressive Growing of GANs for Improved Quality, Stability, and Variation. All three papers are from same set of authors from NVIDIA AI.

Our implementation is a stand-alone minimalistic single GPU Style GAN2 training code. We’ve tried to keep it simple and understandable, and we managed to shrink it to a little more than 350 lines of code.

Generative adversarial networks have two components; the generator and the discriminator. The generator network takes a random latent vector and tries to generate a real looking image. The discriminator network tries to differentiate the real images from generated images. When we train the two together the generator starts generating images indistinguishable from real images.

Progressive GAN

Progressive GAN trains neural networks with adversarial training to generate high-resolution images of size 1028. It does so by progressively increasing the image size. First, it trains a network that produces a 4x4 image, then an 8x8 image, then an 16x16 image, and so on.

At each resolution the generator network produces an image in latent space which is converted into RGB, with a 1x1 convolution. When we progress from a lower resolution to a higher resolution (say from 16x16 to 32x32) we scale the latent image by 2X and add a new block (two 3x3 convolution layers) and a new 1x1 layer to get RGB. The transition is done smoothly by adding a residual connection to the 2X scaled 16x16 RGB image. The weight of this residual connection is slowly reduced, to let the new block to take over.

The discriminator is a mirror image of the generator network. The progressive growing of the disciminator is done similarly.

Image

They use something called equalized learning rates which is discussed below in the implementation. They also use pixel-wise normalization where at each pixel the feature vector is normalized. They apply this to all the convlution layer outputs (except RGB).

Style GAN

Style GAN improves the generator of Progressive GAN keeping the discriminator architecture same.

First it maps the random latent vector ($z \in \mathcal{Z}$) into a different latent space ($w \in \mathcal{W}$), with a 8-layer neural network.

76class MappingNetwork(nn.Module):

77    def __init__(self, features: int, n_layers: int):

92class Discriminator(nn.Module):
# 
93    def __init__(self, n_layers, n_image_features=3, n_features=32, max_features=512):
94        super().__init__()
95
96        self.from_rgb = nn.Sequential(
97            EqualizedConv2d(n_image_features, n_features, 1),
98            nn.LeakyReLU(0.2, True),
99        )
100
101        blocks = []
102        out_features = n_features
103        for i in range(n_layers):
104            in_features = out_features
105            out_features = min(out_features * 2, max_features)
106            blocks.append(DiscriminatorBlock(in_features, out_features))
107
108        self.blocks = nn.Sequential(*blocks)
109
110        self.conv = EqualizedConv2d(out_features, out_features, 4)
111        self.logits = EqualizedLinear(out_features, 1)
# 
113    def __call__(self, x: torch.Tensor):
114        x = self.from_rgb(x)
115        x = self.blocks(x)
116
117        x = self.conv(x)
118        x = x.reshape(x.shape[0], -1)
119        x = self.logits(x)
120        return x
# 
123class DiscriminatorBlock(nn.Module):
# 
124    def __init__(self, in_features, out_features):
125        super().__init__()
126        self.residual = EqualizedConv2d(in_features, out_features, kernel_size=1, stride=2)
127
128        self.block = nn.Sequential(
129            EqualizedConv2d(in_features, in_features, kernel_size=3, padding=1),
130            nn.LeakyReLU(0.2, True),
131            EqualizedConv2d(in_features, out_features, kernel_size=3, padding=1),
132            nn.LeakyReLU(0.2, True),
133        )
134
135        self.down_sample = DownSample(out_features)
136        self.scale = 1 / math.sqrt(2)
# 
138    def forward(self, x):
139        residual = self.residual(x)
140        x = self.block(x)
141        x = self.down_sample(x)
142
143        return (x + residual) * self.scale
# 
146class Generator(nn.Module):
# 
147    def __init__(self, n_layers, d_latent, n_features=32, max_features=512):
148        super().__init__()
149
150        blocks = []
151        in_features = n_features
152        for i in range(n_layers):
153            out_features = in_features
154            in_features = min(in_features * 2, max_features)
155            blocks.append(GeneratorBlock(d_latent, in_features, out_features))
156
157        self.blocks = nn.ModuleList(list(reversed(blocks)))
158
159        self.up_sample = UpSample()
160        self.initial_constant = nn.Parameter(torch.randn((1, in_features, 4, 4)))
# 
162    def forward(self, styles: torch.Tensor, input_noise: torch.Tensor):
163        batch_size = styles.shape[0]
164
165        x = self.initial_constant.expand(batch_size, -1, -1, -1)
166
167        rgb = None
168
169        for i in range(len(self.blocks)):
170            if i != 0:
171                x = self.up_sample(x)
172
173            x, rgb = self.blocks[i](x, rgb, styles[:, i], input_noise)
174
175        return rgb

178class GeneratorBlock(nn.Module):
# 
179    def __init__(self, latent_dim, in_features, out_features):
180        super().__init__()
181        self.to_style1 = EqualizedLinear(latent_dim, in_features)
182        self.to_noise1 = EqualizedLinear(1, out_features)
183        self.conv1 = Conv2dWeightModulate(in_features, out_features, kernel_size=3)
184
185        self.to_style2 = EqualizedLinear(latent_dim, out_features)
186        self.to_noise2 = EqualizedLinear(1, out_features)
187        self.conv2 = Conv2dWeightModulate(out_features, out_features, kernel_size=3)
188
189        self.activation = nn.LeakyReLU(0.2, True)
190        self.to_rgb = ToRGB(latent_dim, out_features)
# 
192    def forward(self, x: torch.Tensor, prev_rgb: torch.Tensor, istyle: torch.Tensor, inoise: torch.Tensor):
193        inoise = inoise[:, :x.shape[2], :x.shape[3], :]
194        noise1 = self.to_noise1(inoise).permute((0, 3, 2, 1))
195        noise2 = self.to_noise2(inoise).permute((0, 3, 2, 1))
196
197        style1 = self.to_style1(istyle)
198        x = self.conv1(x, style1)
199        x = self.activation(x + noise1)
200
201        style2 = self.to_style2(istyle)
202        x = self.conv2(x, style2)
203        x = self.activation(x + noise2)
204
205        rgb = self.to_rgb(x, prev_rgb, istyle)
206        return x, rgb
# 
209class ToRGB(nn.Module):
# 
210    def __init__(self, latent_dim, in_features):
211        super().__init__()
212        self.input_channel = in_features
213        self.to_style = EqualizedLinear(latent_dim, in_features)
214
215        self.conv = Conv2dWeightModulate(in_features, 3, kernel_size=1, demodulate=False)
216        self.up_sample = UpSample()
# 
218    def forward(self, x, prev_rgb, istyle):
219        style = self.to_style(istyle)
220        rgb = self.conv(x, style)
221
222        if prev_rgb is not None:
223            rgb = rgb + self.up_sample(prev_rgb)
224
225        return rgb

228class Conv2dWeightModulate(nn.Module):
# 
229    def __init__(self, in_features, out_features, kernel_size, demodulate=True, lr_mul=1., eps=1e-8):
230        super().__init__()
231        self.filters = out_features
232        self.demodulate = demodulate
233        self.kernel = kernel_size
234        self.padding = (self.kernel - 1) // 2
235
236        he_std = 1 / math.sqrt(in_features * kernel_size * kernel_size)
237        self.weight = nn.Parameter(torch.randn((out_features, in_features, kernel_size, kernel_size)) / lr_mul)
238        self.eps = eps
239
240        self.runtime_coef = lr_mul * he_std
# 
242    def forward(self, x, style):
243        b, c, h, w = x.shape
244
245        w1 = style[:, None, :, None, None]
246        w2 = self.weight[None, :, :, :, :] * self.runtime_coef
247        weights = w2 * (w1 + 1)
248
249        if self.demodulate:
250            d = torch.rsqrt((weights ** 2).sum(dim=(2, 3, 4), keepdim=True) + self.eps)
251            weights = weights * d
252
253        x = x.reshape(1, -1, h, w)
254
255        _, _, *ws = weights.shape
256        weights = weights.reshape(b * self.filters, *ws)
257
258        x = F.conv2d(x, weights, padding=self.padding, groups=b)
259
260        x = x.reshape(-1, self.filters, h, w)
261
262        return x

265class DownSample(nn.Module):
# 
266    def __init__(self, features: int):
267        super().__init__()
268        self.smooth = Smooth()
269        self.conv = EqualizedConv2d(features, features, kernel_size=3, padding=1, stride=2)
# 
271    def forward(self, x: torch.Tensor):
272        return self.conv(self.smooth(x))

275class UpSample(nn.Module):

277        super().__init__()

285class Smooth(nn.Module):
# 
286    def __init__(self):
287        super().__init__()
288        kernel = [[1, 2, 1],
289                  [2, 4, 2],
290                  [1, 2, 1]]
291        kernel = torch.tensor([[kernel]], dtype=torch.float)
292        kernel /= kernel.sum()
293        self.kernel = nn.Parameter(kernel, requires_grad=False)
# 
295    def forward(self, x: torch.Tensor):
296        b, c, h, w = x.shape
297        x = x.view(-1, 1, h, w)
298
299        x = F.conv2d(x, self.kernel, padding=1)
300
301        return x.view(b, c, h, w)

304class EqualizedLinear(nn.Module):
# 
305    def __init__(self, in_features: int, out_features: int, lr_mul: float = 1.):
306        super().__init__()
307        self.weight = nn.Parameter(torch.randn(out_features, in_features) / lr_mul)
308        self.bias = nn.Parameter(torch.zeros(out_features))
309
310        he_std = 1 / math.sqrt(in_features)
311        self.runtime_coef = lr_mul * he_std
312        self.lr_mul = lr_mul
# 
314    def __call__(self, x: torch.Tensor):
315        return F.linear(x, self.weight * self.runtime_coef, bias=self.bias * self.lr_mul)
# 
318class EqualizedConv2d(nn.Module):
# 
319    def __init__(self, in_features: int, out_features: int,
320                 kernel_size: int, padding: int = 0, stride: int = 1,
321                 lr_mul: float = 1.):
322        super().__init__()
323        self.stride = stride
324        self.padding = padding
325        self.weight = nn.Parameter(torch.randn((out_features, in_features, kernel_size, kernel_size)) / lr_mul)
326        self.bias = nn.Parameter(torch.zeros(out_features))
327
328        he_std = 1 / math.sqrt(in_features * kernel_size * kernel_size)
329        self.runtime_coef = lr_mul * he_std
330        self.lr_mul = lr_mul
# 
332    def forward(self, x: torch.Tensor):
333        return F.conv2d(x, self.weight * self.runtime_coef, bias=self.bias * self.lr_mul,
334                        padding=self.padding, stride=self.stride)

337class PathLengthPenalty(nn.Module):
# 
338    def __init__(self, beta: float):
339        super().__init__()
340        self.beta = beta
341        self.steps = nn.Parameter(torch.tensor(0.), requires_grad=False)
342        self.moving_average = nn.Parameter(torch.tensor(0.), requires_grad=False)
# 
344    def forward(self, styles: torch.Tensor, images: torch.Tensor):
345        device = images.device
346        num_pixels = images.shape[2] * images.shape[3]
347        pl_noise = torch.randn(images.shape, device=device) / math.sqrt(num_pixels)
348        outputs = (images * pl_noise).sum()
349
350        gradients, *_ = torch.autograd.grad(outputs=outputs,
351                                            inputs=styles,
352                                            grad_outputs=torch.ones(outputs.shape, device=device),
353                                            create_graph=True)
354
355        path_lengths = (gradients ** 2).sum(dim=2).mean(dim=1).sqrt()
356
357        if self.steps > 0:
358            a = self.moving_average / (1 - self.beta ** self.steps)
359            loss = torch.mean((path_lengths - a) ** 2)
360        else:
361            loss = path_lengths.new_tensor(0)
362
363        mean = path_lengths.mean().detach()
364
365        if not torch.isnan(mean):
366            self.moving_average.mul_(self.beta).add_(mean, alpha=1 - self.beta)
367            self.steps.add(1.)
368
369        return loss

471    def z_noise(self, batch_size):

482        z_style = self.z_noise(batch_size)
483        w_style = self.z_to_w(z_style)

490    def step(self, idx):
#

Train Discriminator

492        self.discriminator_optimizer.zero_grad()
493
494        for i in range(self.gradient_accumulate_steps):
495            generated_images, _ = self.generate_images(self.batch_size)
496            fake_output = self.discriminator(generated_images.detach())
497
498            x = next(self.loader).to(self.device)
499            x.requires_grad_()
500            real_output = self.discriminator(x)
501
502            real_loss, fake_loss = self.discriminator_loss(real_output, fake_output)
503            disc_loss = real_loss + fake_loss
504
505            if (idx + 1) % self.lazy_gradient_penalty_interval == 0:
506                gp = self.gradient_penalty(x, real_output)
507                tracker.add('loss.gp', gp)
508                disc_loss = disc_loss + gp
509
510            disc_loss.backward()
511
512            tracker.add('loss.discriminator', disc_loss)
513
514        self.discriminator_optimizer.step()
#

Train Generator & Mapping Network

517        self.generator_optimizer.zero_grad()
518        self.mapping_network_optimizer.zero_grad()
519
520        for i in range(self.gradient_accumulate_steps):
521            generated_images, w_style = self.generate_images(self.batch_size)
522            fake_output = self.discriminator(generated_images)
523
524            gen_loss = self.generator_loss(fake_output)
525
526            if idx > self.lazy_path_penalty_after and (idx + 1) % self.lazy_path_penalty_interval == 0:
527                ppl = self.path_length_penalty(w_style, generated_images)
528                if not torch.isnan(ppl):
529                    gen_loss = gen_loss + ppl
530
531            gen_loss.backward()
532
533            tracker.add('loss.generator', gen_loss)
534
535        self.generator_optimizer.step()
536        self.mapping_network_optimizer.step()
537
538        if (idx + 1) % self.log_generated_interval == 0:
539            tracker.add('generated', generated_images)
540        if (idx + 1) % self.save_checkpoint_interval == 0:
541            experiment.save_checkpoint()
